# 🚀 CodeCraft — Learn Programming Faster

A modern SaaS-style landing page for a fictional coding education platform, inspired by products like Linear, Vercel, and Notion.

Built with a sleek **Slate + Lavender** dark theme, smooth animations, responsive layouts, and interactive UI elements to showcase modern frontend development practices.

> **Note:** CodeCraft is a fictional project created for frontend development and portfolio purposes. All content, statistics, testimonials, reviews, and platform information displayed on the website are placeholder data and do not represent a real product, company, or service.

---

## ✨ Features

* Responsive design for mobile, tablet, and desktop
* Fixed navbar with blur and shadow effects on scroll
* Animated mobile navigation menu
* Modern hero section with gradient glow effects
* Interactive feature cards with hover animations
* Split-layout About section
* Animated statistics counters using Intersection Observer
* Testimonials section with review cards
* Contact form with client-side validation
* Custom scrollbar styling
* Custom cursor with hover interactions (desktop only)
* Scroll-triggered animations powered by AOS

## ⚡ Getting Started

Clone the repository and open the project locally:

```bash
git clone https://github.com/Jofil-Joby/SCT_WD_1.git
cd codecraft
```

Open `index.html` in your browser.

Or run a local server:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```
